1. Download Pointer app.
2. Add API Keys
3. Modify tools, rules, and MCP servers to customize your agent
4. Start a new session and begin automating tasks 

![Pointer Ambient Agent](/images/pointerambient.png)

The Pointer ambient agent runs continuously in the background, ready to assist with tasks across your desktop environment. It provides seamless integration with your workflow while maintaining full control over when and how it operates.

![Pointer Agent Interface](/images/pointergui.png)

The agent interface allows you to configure tools, set custom rules, and manage MCP server connections to tailor the agent's capabilities to your specific needs.
