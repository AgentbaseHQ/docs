Choose your preferred way to interact with Agentbase's general-purpose AI agent and get started instantly.

### Agent API (For Developers)
Integrate Agentbase's agent directly into your applications.

<AccordionGroup>
  <Accordion title="1. Get Your Computer Environment" icon="key">
    Navigate to your [Computer page](https://agentbase.sh) on the Agentbase dashboard, sign in, and generate a new session.
  </Accordion>
  <Accordion title="2. Make Your First API Call" icon="code-branch">
    Use this cURL command in your terminal to send your first message to the agent.
    ```bash
    curl -X POST "https://api.agentbase.sh" \
      -H "Content-Type: application/json" \
      -H "Accept: text/event-stream" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"message": "What can you do?"}'
    ```
    You'll receive a real-time stream of the agent's response.
  </Accordion>
</AccordionGroup>
