Interact with Agentbase's general-purpose AI agent and get started instantly.

### Agent API

Integrate Agentbase's agent directly into your applications via our Agent API.

<AccordionGroup>
  <Accordion title="1. Get Your API Key" icon="key">
    Navigate to the [Overview page](https://base.agentbase.sh/overview) on the Agentbase dashboard, sign in, and generate a new API key.
  </Accordion>
  <Accordion title="2. Purchase credits" icon="credit-card">
    Navigate to the [Billing page](https://base.agentbase.sh/home/credits) on the Agentbase dashboard and purchase credits.
  </Accordion>
  <Accordion title="3. Make Your First API Call" icon="code">
    Use this cURL command in your terminal to send your first message to the agent.
    ```bash
    curl -X POST "https://api.agentbase.sh" \
      -H "Content-Type: application/json" \
      -H "Accept: text/event-stream" \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"message": "What can you do?"}'
    ```
    You'll receive a real-time stream of the agent's response.
  </Accordion>
</AccordionGroup>
