| Type | Description |
|------|-------------|
| `agent_started` | Session started |

| `agent_response_start` | Beginning of response |
| `agent_response` | Response chunk |
| `agent_tool_use_start` | Tool call started |
| `agent_tool_response` | Tool result |
| `agent_cost` | Cost info |
| `agent_step` | Step completed |
| `agent_completed` | Session finished |
