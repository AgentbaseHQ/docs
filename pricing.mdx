> Pay per action, not per token. Simple pricing that scales with value delivered.

## Cost-per-Step Pricing

*Currency: USD ($)*

| Mode | Context Window Size | Cost per Step |
|:-----|:--------------------|:--------------|
| Flash | 0 - 30K tokens | $0.01 |
| Flash | 30K - 200K tokens | $0.01 - $0.10 |
| Fast | 0 - 30K tokens | $0.05 |
| Fast | 30K - 200K tokens | $0.10 - $0.25 |
| Max | 0 - 30K tokens | $0.20 |
| Max | 30K - 200K tokens | $0.20 - $0.75 |


<Note>
**Extended Context**: We offer long context up to 1M tokens for specialized use cases. Contact us for pricing and availability.
</Note>

<Tip>
**Beta Phase**: During our beta phase, all other resources like computers, datastore, etc are free. You only pay for agent steps.
</Tip>
