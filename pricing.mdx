> Pay per action, not per token. Simple pricing that scales with value delivered.

## Cost-per-Step Pricing

*Currency: USD ($)*

| Mode | Context Window Size | Cost per Step |
|------|-------------------|---------------|
| **Flash** | < 30k tokens | $0.005 |
| | 30k - 128k tokens | $0.012 |
| **Fast** | < 30k tokens | $0.05 |
| | 30k - 128k tokens | $0.12 |
| **Max** | < 30k tokens | $0.15 |
| | 30k - 128k tokens | $0.40 |

<Note>
**Context-Based Pricing**: Costs increase with larger context windows (30k+ tokens) due to higher computational requirements. Most standard workflows use small context (< 30k tokens) for optimal cost efficiency.
</Note>
