## Cost Event Structure

Every API response includes an `agent_cost` event:

```json
{
  "type": "agent_cost",
  "session": "sess_abc123",
  "cost": "0.0091",
  "balance": 49.99,
  "deductionSuccess": true,
  "lowBalance": false
}
```
